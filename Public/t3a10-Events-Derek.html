Events Webpage

This section of the webpage demonstrates onmousemovement, onmouseenter and onmouseover.

The 'onmousemove' event happens whenever the cursor is moved while over the div element.

The 'mouseenter' event only occurs whenever the cursor enters the div element.

The 'onmouseover' event happens whenever the cursor enters the div element, and its child elements (p and span).
